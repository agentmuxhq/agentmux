## Current State

### What Works

- Window: launch, resize, minimize, maximize, drag (frameless + CEF Views)
- Backend sidecar: spawns via Job Object, auto-killed on exit
- IPC: HTTP POST bridge (~40 commands), auth token verified
- Terminal, sysinfo, agent, forge, identity, help panes
- Context menus (JS overlay)
- Pane/tab DnD via pragmatic-dnd
- DevTools: remote debugging on port 9222
- Portable: clean layout (agentmux.exe launcher + runtime/ subdirectory)
- File stripping: SwiftShader, WebGPU, extra locales removed (~30 MB saved)
- Full version isolation: data, config, settings, auth, CLI installs all per-version

1. **Multi-window** — commands exist, not wired to UI
2. **Cross-window drag** — 10 commands stubbed

4. **React cleanup** — dead deps in package.json, some `React.*` types in code

## Key Fix This Session

**Window resize** — `can_resize()`, `can_maximize()`, `can_minimize()` were not overridden in `AgentMuxWindowDelegate`. CEF Views defaults them to 0 (disabled). Fixed by returning 1.

**Version isolation** — Changed from shared `ai.agentmux.cef/` root to per-version `ai.agentmux.cef.v0-32-NNN/` directories, matching Tauri's pattern. Everything isolated: data, config, settings, auth tokens, CLI installs.
