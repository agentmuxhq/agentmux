# Session Handoff — 2026-03-08 19:24 PDT

## Branch: `agenta/pane-zoom-v2` (based on `origin/main`)

## What Was Done

### Per-Pane Zoom (complete, committed)
- Ctrl+/- zooms the **focused** terminal pane via `term:zoom` block metadata
- Ctrl+Scroll zooms the pane **under the mouse cursor** (hover-aware, no focus steal)
- Font size updated in-place via `terminal.options.fontSize` — no full TermWrap reconstruction, no WebGL context churn
- Zoom indicator shows percentage overlay with 1.5s auto-hide

### Chrome Zoom (complete, committed)
- Ctrl+Scroll over title bar or status bar zooms both together
- Uses CSS `zoom` property on `.window-header` and `.status-bar` containers
- Single `--zoomfactor` CSS custom property drives both
- Removed all `--zoomfactor-inv` references (legacy compensator for old Tauri window zoom)
- All hardcoded px values in child elements (tabs, icons, widgets) scale uniformly via CSS `zoom`

### Old Global Window Zoom (removed)
- `window.set_zoom()` locked to 1.0 on startup
- `zoomFactorAtom` (global) removed, replaced with per-block `termZoomAtom`
- `updateZoomFactor()` in wave.ts removed
- `loadZoom()` is now a no-op
