Core Features

Everything you need to orchestrate agents

A visual workspace for monitoring autonomous agents as they build complex work - code, research, reports, and more.

Multi-Agent Workspace

Watch a team of agents work in split panes. Each agent gets its own context, model, and task. Monitor progress across research, code, reports, and analysis simultaneously.

Community

Multi-Model Support

Connect any AI model: Claude, GPT, Gemini, Ollama, or custom endpoints. Bring your own API keys. Switch models per agent instantly.

Community

Agent Messaging

Agents communicate through a built-in message bus. Route context between agents, pass outputs as inputs, and coordinate multi-step workflows.

Community

Remote Agents

Run agents on remote machines via SSH. Manage cloud instances, connect to servers, and orchestrate agents across your entire infrastructure.

Community

Context & File Aware

Agents see your files, terminals, and web views. Preview documents, data, images, and dashboards inline. Every agent has the context it needs.

Community

Built-in Terminal

Full-featured terminal with shell integration, command completion, and wsh CLI. Agents can execute commands, and you can intervene at any point.

Advanced Orchestration

Go beyond running agents in isolation. Pin shared context, @mention other agents inline, branch conversations to explore alternatives, and compare model outputs side by side.

  • Full conversation history with search across all agents
  • Model presets: assign Claude, GPT, Ollama per agent
  • Context pinning: share files across agents persistently
  • @mentions: reference any agent, terminal, or file inline
  • Conversation branching: fork any agent's thread
  • Multi-model compare: same task, different models, side by side

Workflow Automation

Define multi-step agent workflows, record sessions for replay, and build a library of reusable agent configurations. Automate the repetitive parts of orchestration.

  • Session recording and replay (asciinema compatible)
  • Smart command history with frequency ranking
  • Agent templates: save and restore full agent setups
  • Snippet library for reusable prompts and commands

About AgentMux - AgentMux is a private fork of Wave Terminal. While it inherits Wave's terminal foundation, AgentMux's core orchestration systems - agent messaging, multi-model routing, workflow automation, and the inter-agent coordination layer - were designed and built from scratch by AI agents running inside AgentMux itself.
